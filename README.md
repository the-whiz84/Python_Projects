### Prerequisites
- **Python 3.x**: [Download here](https://www.python.org/downloads/)

  ```bash
  python -m venv .venv
  source .venv/bin/activate  # MacOS/Linux

  ```

Cloning the repository and installing dependencies (where applicable):
```bash

```

